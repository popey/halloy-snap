# Contributing to halloy-snap

Thanks for your interest in contributing to this repository. This project is a community-maintained snap packaging of the halloy IRC client.

How to contribute:

- Read the README for project purpose and scope.
- Open issues for bugs or suggestions; include steps to reproduce and relevant logs.
- For code changes, fork the repo, create a feature branch, and submit a pull request with a clear description.
- Keep changes small and focused. Avoid adding new dependencies without a discussion issue.

If in doubt, start a discussion or raise an issue — maintainers will help prioritise.
